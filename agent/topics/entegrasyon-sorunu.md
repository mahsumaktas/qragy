# Entegrasyon / API Sorunu

Kullanici dis sistem entegrasyonunda veya API baglantisinda sorun yasiyor.

## Ilk Kontrol Adimlari
1. API anahtarinin dogru ve gecerli oldugunu kontrol edin.
2. Entegrasyon ayarlarinda URL ve endpoint bilgilerini dogrulayin.
3. Dis sistemin aktif ve erisilebilir oldugunu kontrol edin.
4. Hata mesajini not edin ve destek ekibine iletin.

## Yaygin Sorunlar
API key hatasi: Anahtar suresi dolmus veya yanlis girilmis olabilir.
Baglanti zaman asimi: Dis sistemin erisilebirligi kontrol edilmeli.
Veri senkronizasyon hatasi: Veri formati uyumsuzlugu olabilir.

## Bot ne yapmali
Hata mesajini sor — bu escalation icin kritik bilgi.
API key ve endpoint dogrulamasini oncelikli oner.
Bu tarz sorunlar genellikle teknik destek gerektirir, escalation'a erken yonlendir.

## Bot ne YAPMAMALI
API konfigurasyonu yapmaya calisma — bot sistem ayari degistiremez.
Dis sistem hakkinda tahmin yurutme.
Teknik detaylara (HTTP status kodlari, header vb.) girme.

## Eskalasyon
Entegrasyon sorunlari teknik destek gerektirir. Gerekli bilgi:
kullanici_adi: Sorun yasayan kullanicinin adi.
