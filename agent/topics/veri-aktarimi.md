# Veri Aktarimi (Import/Export)

Kullanici veri yukleme (import) veya indirme (export) islemi yapmak istiyor veya sorun yasiyor.

## Import Adimlari
1. Yuklenecek dosyanin desteklenen formatta oldugunu kontrol edin (CSV, Excel).
2. Dosya basliklarinin sistem tarafindan beklenen basliklarla uyumlu oldugunu dogrulayin.
3. Dosya boyutunun limite uygun oldugunu kontrol edin.
4. Yukleme sayfasindan dosyayi secin ve "Yukle" butonuna tiklayin.

## Export Adimlari
1. Indirmek istediginiz veri sayfasina gidin.
2. Disa aktar / Export butonuna tiklayin.
3. Format secin (CSV veya Excel).
4. Indirme islemini bekleyin.

## Yaygin Sorunlar
Dosya yuklenemiyor: Format veya boyut limiti kontrol edin.
Veriler eksik gorunuyor: Filtre ayarlarini kontrol edin.
Karakter sorunu: Dosyanin UTF-8 formatinda kaydedildiginden emin olun.

## Bot ne yapmali
Import sorunlarinda dosya formati ve baslik uyumunu kontrol etmesini oner.
Export sorunlarinda filtre ayarlarini ve tarayici indirme ayarlarini kontrol etmesini oner.
Bu konu genellikle kullanici tarafinda cozulebilir.
